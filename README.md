# Aplikasi Manajemen Pengajian Yayasan Sahabat Quran Bogor #

Tech Stack:

* Java 21
* Spring Boot 3.3.3
* PostgreSQL 16

Dev Stack:

* Maven 3
* Docker Compose
* TestContainer
* Selenium
* Jacoco Coverage Test
* Sonarcloud Quality

## Daftar Fitur ##

### Admin ###

* Data Pengajar
* Data Bank dan Rekening
* Data Kurikulum
* Data Ruangan
* Data Kelas
* Plotting Kurikulum ke Kelas
* Plotting Kelas ke Pengajar
* Plotting Kelas ke Ruangan
* Jadwal Belajar
* Jadwal Ujian
* Soal Ujian
* Parameter penilaian ujian / bobot penilaian
* Event

## Finance ##

* Setting Nilai Iuran
* Generate Tagihan Iuran
* Pembayaran Iuran (Manual dan VA)
* Infaq non Iuran

## Peserta ##

* Registrasi
* Lihat Tagihan
* Pembayaran Iuran
* Jadwal Belajar
* Input Mutabaah (catatan kemajuan pelajaran)
* Jadwal Event
* Input Presensi Kelas
* Input Presensi Event
* Laporan Hasil Belajar

## Pengajar ##

* Daftar Kelas yang dipegang
* Jadwal Mengajar
* Start sesi belajar
* Input presensi peserta
* Finish sesi belajar
* Input berita acara sesi pelajaran
* Lihat mutabaah peserta
* Start sesi ujian praktek
* Input nilai ujian praktek
* Finish sesi ujian praktek
* Rekap sesi pengajaran
* Slip Gaji
* Rekap sesi ujian
